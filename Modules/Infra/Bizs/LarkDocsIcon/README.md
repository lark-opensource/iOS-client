## 组件作用

1.LarkDocsIcon主要基于 CCM 图标显示能力，进行下沉的基础组件，负责收敛 Lark所有显示文档icon 的能力。
2.支持显示3种类型图标
- 根据文档类型显示默认图标
- 自定义emoji图标
- 自定义icon图片图标
