# SKBitable

## 代码架构

### 文件结构
src
│── Interface ...........................<接口层>
│   └── BitableModule.swift
├── Application .........................<应用层>
│   ├── BitableBrowser ....................<Base 主文档容器>
│   ├── FieldEditor .......................<字段 Meta 编辑>
│   ├── Forms .............................<收集表>
│   ├── Homepage ..........................<Tab 主页>
│   └── ItemView ..........................<单条记录>
├── Service .............................<服务层>
│   ├── API ...............................<网络>
│   ├── Cache .............................<缓存>
│   ├── JSService .........................<JS 服务>
│   ├── Lynx ..............................<Lynx>
│   ├── Permission ........................<权限>
│   ├── Statistic .........................<埋点>
│   ├── Third .............................<第三方业务>
│   └── UIComponent .......................<公共业务 UI 组件>
│       ├── DDUI ............................<Data Defined UI>
│       ├── EventMonitor ....................<UIEvent Hook>
│       ├── Panel ...........................<弹出面板>
│       └── Router ..........................<导航路由>     
└── Foundation ..........................<基建层>
    ├── Extension .........................<工具扩展>
    ├── UIWidget ..........................<通用 UI 组件>
    └── Util ..............................<基础工具>
